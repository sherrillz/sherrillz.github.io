INTRODUCTION TO GIT AND GITHUB

Learning Git and Github

Shell commands

  1. pwd know the path of the folder
  2. ls know the files under the folder
  3. cd changing directory of the files 
    cd subfolder
    The file will be changed to the subfolder under the same path
  4. nano edit a file: delete, add or change content
    1. ctrl+0 save changes
    2. ctrl+x exit editor
  5. Editing a file
    1. echo—create or edit a file
    2. Create a new file todo.txt
       echo "Review for duplicate records" > todo.txt
  1. Add content to existing file todo.tx
   echo "Review for duplicate records" >> todo.txt
