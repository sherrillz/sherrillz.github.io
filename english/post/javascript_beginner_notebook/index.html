Introduction

JavaScript is a versatile, beginner-friendly language used for web development, server-side scripting, and more. In this notebook, you’ll learn core concepts and write your first lines of JavaScript.

Setting Up Environment

  1. Web Browser: Modern browsers (Chrome, Firefox, Edge) include a JavaScript console.
  2. Code Editor: Install VS Code, Sublime Text, or another editor.
  3. Live Server (optional): VS Code extension to preview HTML/JS.

Create a new folder:

mkdir js-notebook
cd js-notebook

Inside, create an index.html and a script.js.

index.html

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>JS Notebook</title>
</head>
<body>
  <h1>JavaScript Beginner Notebook</h1>
  <script src="script.js"></script>
</body>
</html>

Hello, World!

Open script.js and add:

console.log('Hello, World!');
  1. Open your browser’s console (F12 or Ctrl+Shift+I).
  2. Refresh the page. You should see Hello, World! logged.

Variables & Data Types

JavaScript variables store values. Use let, const, or var (avoid var in modern code).

let name = 'Alice';       // string
const age = 30;           // number
let isStudent = false;    // boolean
let score;                // undefined
let value = null;         // null

Try changing values and logging:

console.log(name, age, isStudent);

Operators

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, -=
  • Comparison: ==, !=, ===, !==, <, >
  • Logical: &&, ||, !

Example:

let x = 10;
x += 5;        // x = 15
console.log(x > 10 && x < 20); // true

Control Flow

If / Else

let temperature = 75;
if (temperature > 80) {
  console.log('Hot');
} else if (temperature < 60) {
  console.log('Cold');
} else {
  console.log('Comfortable');
}

Loops

  • for 
    for (let i = 0; i < 5; i++) {
  console.log(i);
}
  • while 
    let count = 0;
while (count < 3) {
  console.log(count);
  count++;
}

Functions

Define reusable code blocks.

function greet(person) {
  return `Hello, ${person}!`;
}
console.log(greet('Bob')); // Hello, Bob!

Arrow functions (ES6):

const add = (a, b) => a + b;
console.log(add(2, 3)); // 5

Arrays

Store lists of values.

let colors = ['red', 'green', 'blue'];
colors.push('yellow');
console.log(colors[2]); // blue

Common methods: push, pop, shift, unshift, forEach.

colors.forEach(c => console.log(c));

Objects

Key-value pairs for structured data.

let person = {
  name: 'Charlie',
  age: 25,
  hobbies: ['reading', 'gaming']
};
console.log(person.name);
person.job = 'Developer';

Loop through keys:

for (let key in person) {
  console.log(key, person[key]);
}

DOM Manipulation

Interact with the page content.

In index.html, add:

<button id="btn">Click me</button>
<p id="message"></p>

In script.js:

const btn = document.getElementById('btn');
const msg = document.getElementById('message');

btn.addEventListener('click', () => {
  msg.textContent = 'Button clicked!';
});

Happy coding! 🎉
